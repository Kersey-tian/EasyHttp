# EasyHttp
基于 okhttp 网络请求框架，上传下载文件，断点续传

##使用方法

Download the latest JAR or grab via Maven:

    <dependency>
      <groupId>com.lz.easyhttp</groupId>
      <artifactId>easyhttp</artifactId>
      <version>0.1.0</version>
    </dependency>
    
or Gradle:

    compile 'com.lz.easyhttp:easyhttp:0.1.0'
